# Gestion Biblio — Projet étudiant

##  Description
Gestion Biblio est un projet étudiant développé en JavaScript permettant de gérer
un inventaire de livres en local à l’aide du navigateur (localStorage).
Ce projet vise à appliquer de bonnes pratiques de développement et de structuration
de projet.

##  Objectifs pédagogiques
- Concevoir une application simple et fonctionnelle
- Structurer un projet selon des standards clairs
- Utiliser Git et GitHub pour le versionnement
- Documenter un projet de manière professionnelle
- Mettre en place des tests de base

##  Technologies utilisées
- JavaScript
- HTML
- CSS
- Git / GitHub
- LocalStorage (stockage côté client)

##  Structure du projet
gestion_biblio/
│── src/ # Code source principal
│── tests/ # Tests du projet
│── docs/ # Documentation
│── assets/ # Ressources (images, fichiers)
│── .gitignore # Fichiers ignorés par Git
│── README.md # Documentation du projet


##  Fonctionnalités principales
- Ajouter un livre à l’inventaire
- Supprimer un livre
- Afficher la liste des livres
- Sauvegarde automatique des données en local

##  Utilisation
1. Cloner le dépôt
2. Ouvrir le fichier principal HTML dans un navigateur
3. Utiliser l’interface pour gérer l’inventaire de livres

##  Contexte académique
Ce projet a été réalisé dans le cadre d’un cours académique.
Il sert de démonstration des compétences acquises en développement,
en organisation de projet et en utilisation de Git.
