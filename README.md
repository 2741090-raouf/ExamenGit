# ExamenGit

## Description
Ce projet a été réalisé dans le cadre d’un examen sur Git et GitHub.
Il démontre l’utilisation des concepts de base de Git.

## Contenu du projet
- index.html : page HTML simple utilisée pour les tests de versionnement.

## Technologies utilisées
- HTML
- Git
- GitHub

## Objectif pédagogique
- Comprendre le workflow Git (add, commit, push)
- Utiliser un dépôt GitHub
- Gérer les versions d’un projet simple
